Accessibility

What is it like to live with a handicap? What is it like to be unable to use an app or website? To answer these questions I have started with creating short movies about people. I want to know what it is like and I want to show it to you. Hopefully in this way you will, like me, gain a better understanding and it might help you creating better software that can be used by everyone. Because accessibility is about...
